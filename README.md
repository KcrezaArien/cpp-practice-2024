# cpp-practice
Collection of beginner C++ exercises covering core programming concepts such as loops, conditionals, arrays, strings, and basic algorithms (factorial, Fibonacci, prime check, palindrome, etc.).

## 📚 Topics Covered
* Basic input/output
* Conditionals (if/else)
* Loops
* Arrays
* Strings
* Basic algorithms

## 🧠 Exercises Included
1. Hello World – Print “Hello, World!” to the console
2. Sum of Two Numbers – Add two numbers and display the result
3. Factorial of a Number – Compute factorial using iteration/recursion
4. Even or Odd – Check if a number is even or odd
5. Reverse a String – Reverse characters in a string
6. Fibonacci Series – Generate Fibonacci sequence
7. Prime Number Check – Determine if a number is prime
8. Find Maximum Element – Find the largest value in an array
9. Palindrome Check – Check if a string reads the same forward/backward
10. Find Minimum Element – Find the smallest value in an array

## 🎯 Purpose
This repository serves as:
* Practice for improving C++ fundamentals
* A reference for basic programming problems
* A way to track my learning progress

## 📌 Notes
These are beginner-level exercises focused on clarity and understanding rather than optimization.
